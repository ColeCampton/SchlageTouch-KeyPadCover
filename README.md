# Schlage Touch Keypad Rain Cover (Openscad 3d Model & STL)

Here is a design file and printable files for a rain cover for the [Schlage Touch Lock](https://www.schlage.com/en/home/products/BE375CENFFF.html).

This features slots for four [2x2mm magnets](https://www.amazon.com/dp/B07CSM2G9K) to be press-fit for holding the cover closed, and four screw holes for mounting.
